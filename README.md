Deep Learning based image classification system that identifies medicinal plants using leaf images.

---

## 🚀 Key Features
- Image classification using CNN
- Detect plant species from leaf images
- Image preprocessing and training pipeline
- Model accuracy evaluation
- Predict class for new images

---

## 🛠 Tech Stack
Python • TensorFlow • Keras • NumPy • Pandas • Matplotlib
